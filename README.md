🕶️ Shree Swami Opticals – Optical Store Website

Shree Swami Opticals is a modern and responsive web-based optical store designed to provide users with an easy and interactive experience while browsing and purchasing eyewear online. The platform allows users to explore a wide range of glasses, view detailed product information, and virtually try on frames before making a purchase.

The website focuses on user-friendly design, mobile responsiveness, and interactive features similar to professional optical platforms like LensKart.

🚀 Key Features
👓 Product Catalog

Displays multiple eyewear products including frames and glasses.

Each product includes image, name, price, and detailed description.

Users can view detailed product pages for each item.

🧑‍💻 Virtual Try-On

Users can try glasses virtually using their device camera.

Glasses automatically adjust according to the user’s face.

Provides a realistic preview before purchase.

🛒 Shopping Cart

Users can add products to the cart from the product page.

Cart system manages selected products before checkout.

🔐 User Authentication

Secure user Signup and Login system using Firebase Authentication.

Allows users to access personalized features after login.

📱 Fully Responsive Design

Optimized for mobile phones, tablets, and desktops.

Clean UI inspired by professional e-commerce optical platforms.

💬 Chatbot Support

Integrated chatbot in the user dashboard.

Helps users with product information and assistance.

🛠️ Technologies Used

HTML5 – Structure of the website

CSS3 – Styling and responsive design

JavaScript (Vanilla JS) – Dynamic functionality

Firebase Authentication – User login/signup system

Firebase Database / Storage – Data management

Face Detection / Camera API – Virtual try-on functionality

🎯 Project Objective

The main objective of this project is to build a smart and interactive optical store platform that enhances the online shopping experience by allowing users to visualize eyewear using a virtual try-on system and providing a smooth, responsive, and modern interface.
